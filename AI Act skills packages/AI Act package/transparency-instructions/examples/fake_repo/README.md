# ArrhythmiaDetector

**Version**: 2.1.0
**Provider**: MediTech Solutions GmbH

## Intended Purpose

This software is intended to be used by qualified cardiologists to assist in the diagnosis of cardiac arrhythmia. It analyzes 12-lead ECG data.

## Hardware Requirements

- NVIDIA GPU with 8GB VRAM
- 16GB RAM
- FDA-approved ECG capture device

## Limitations

- Not for use on pediatric patients (<18 years).
- Accuracy reduced in high-noise environments.
