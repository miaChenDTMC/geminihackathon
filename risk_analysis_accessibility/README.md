# Accessibility Risk Analysis Module

## Description
This module performs automated accessibility auditing for High-Risk AI systems.
It checks for compliance with WCAG 2.1 standards and analyzes violations against
Fundamental Rights (Article 27 / EU Charter Art 26).

## Tech Stack
- Selenium (Headless Browser)
- Axe-core (Accessibility Engine)
- Gemini Pro (Risk Interpretation)
